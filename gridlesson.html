Grid

Two-dimensional layout mastery · rows & columns reimagined
§ 3.1 — What is Grid?

Grid is a powerful CSS layout system used to arrange elements in rows AND columns simultaneously.

👉 Think of Grid like a sophisticated table layout — but far more flexible for modern web design.

🔹 Unlike Flexbox (one-dimensional), Grid handles both axes with ease.
§ 3.2 — Basic Setup
.container {
  display: grid;
}

→ Turning an element into a grid container unlocks grid properties for its direct children (grid items).

§ 3.3 — Creating Columns
grid-template-columns: 1fr 1fr 1fr;

👉 This creates 3 equal columns using the fr unit (fractional unit). The available space is divided equally.

/* other examples: */
grid-template-columns: 200px 1fr 2fr; /* mixed units */
grid-template-columns: repeat(4, 1fr); /* shorthand for 4 columns */
§ 3.4 — Creating Rows
grid-template-rows: 100px 100px;

Explicitly defines the height of rows. In this case: two rows, each 100px tall. Items will fill into these row tracks automatically.

🔸 Notice explicit 100px row height. Great for dashboards or strict vertical rhythm.
§ 3.5 — Spacing Between Items
gap: 10px;

Sets both row-gap and column-gap in one declaration. You can also use column-gap and row-gap individually.

⬆️ Example uses gap: 20px 12px (row-gap 20px, column-gap 12px). The classic gap:10px gives uniform spacing.

§ 3.7 — Making an Item Bigger (Spanning Columns)
.box1 {
  grid-column: 1 / 3;
}

👉 This makes the item span across columns 1 through 3 (column line 1 to line 3). The item will occupy two column tracks. Perfect for hero elements or featured content.

Featured item (spans col 1 to 3)
Item 2
Item 3
⭐ The first box uses grid-column: 1 / 3; → expands across 2 columns, while remaining items auto-place into the next grid cells.
/* additional spanning examples */
.item { grid-column: 2 / 4; } /* span from column line 2 to 4 */
.item { grid-row: 1 / 3; } /* span rows as well */
.hero { grid-column: 1 / -1; } /* span all columns */
§ 3.8 — When to Use Grid

🎯 Use CSS Grid when:

  • 🏛️ You are designing a whole page — macro layouts, magazine-style, application shell
  • 🧩 You need rows and columns simultaneously — both dimensions controlled at once
  • 📐 You want a structured layout — precise alignment, overlapping zones, asymmetric grids

✨ Pro tip: Use Grid for page skeleton (header, sidebar, main, footer) and Flexbox for inner components. Together they’re unbeatable.
