How to Host a Simple Webpage on GitHub

Step 1: Create a GitHub Account

  1. Go to https://github.com
  2. Click Sign Up
  3. Enter your username, email, and password
  4. Verify your account

Step 2: Create a Repository

  1. Click the + icon → New repository
  2. Name it: yourusername.github.io Example: juan143.github.io
  3. Set it to Public
  4. Click Create repository

Step 3: Upload Your Files

  1. Click Add file → Upload files
  2. Upload your index.html or any html file
  3. You can also add other files (CSS, JS, images)
  4. Click Commit changes

Step 4: Enable GitHub Pages

  1. Go to Settings
  2. Click Pages
  3. Select branch: main
  4. Folder: / (root)
  5. Click Save

Step 5: View or Share Your Page

You can view or share your page using the link:

https://yourusername.github.io/filename

Example:

https://juan143.github.io/mypage.html
