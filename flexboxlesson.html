Flexbox

Flexible Box Layout · one-dimensional precision & alignment
§ 2.1 — What is Flexbox?

Flexbox is a layout module used to arrange elements in one direction only:

🔹 Either row (horizontal) — left to right
🔹 Or column (vertical) — top to bottom

👉 Think of Flexbox as arranging items in a straight line — like a flexible, responsive ribbon.

✨ Perfect for UI components, navigation bars, card decks, and centering elements.

§ 2.4 — Aligning Items (Very Important)

a. Horizontal Alignment

justify-content: center;

Other values:

  • flex-start → left (default)
  • flex-end → right
  • space-between → space in between items
  • space-around → equal space around each item
  • space-evenly → perfectly distributed gaps

b. Vertical Alignment

Other values:

  • flex-start → top
  • flex-end → bottom
  • stretch → fill container height (default)

§ 2.5 — Making Items Equal Size
.box {
  flex: 1;
}

👉 All items will have equal width (or equal height if column direction). The flex: 1 property distributes extra space equally.

🌟 Each item gets flex: 1 → all take equal share of available space.
/* Advanced: flex: 2 (twice the size) */
.item-large { flex: 2; }
.item-small { flex: 1; }

§ 2.7 — When to Use Flexbox

🎯 Use Flexbox when:

  • 📏 You want items in a row (or column) — one-dimensional alignment
  • 🎯 You want to center something — vertically + horizontally (with justify + align)
  • 🍔 You are making menus or navigation — responsive navbars, tabs, toolbars
  • 📦 Building card decks, form layouts, or action bars — distributing space dynamically

✨ Pro insight: Flexbox is your go‑to for components; Grid is for overall page structure. Combine both for mastery.
